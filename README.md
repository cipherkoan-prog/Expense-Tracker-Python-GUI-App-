# 💰 Expense Tracker (Python GUI App)

A simple and powerful **Expense Tracker application** built using Python.
This project helps users manage their daily expenses with a clean graphical interface.

---

## 🚀 Features

* ➕ Add expenses (amount + category + date)
* 📋 View all expenses in a list
* ❌ Delete selected expenses
* 🔍 Filter expenses by category
* 📅 Monthly report (filter by month)
* 💰 Show total expenses
* 📊 Visualize expenses using charts
* 📁 Export data to Excel file
* 💾 Data stored permanently using JSON

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/expense-tracker.git
```

2. Navigate to the project folder:

```
cd expense-tracker
```

3. Install required libraries:

```
pip install matplotlib openpyxl
```

4. Run the application:

```
python ExpenseTracker.py
```
